# CNA Innovation Score Dashboard

This Streamlit app evaluates Certified Nursing Assistants (CNAs) across various performance and innovation metrics.
It calculates a weighted innovation score and classifies CNAs into performance tiers.

## Features
- Real-time scoring
- Tier classification (Aâ€“F)
- Easy web deployment with Streamlit Cloud

## Run Locally
```
pip install -r requirements.txt
streamlit run cna_score_dashboard.py
```

## Deploy on Streamlit Cloud
1. Fork this repo.
2. Log into https://streamlit.io/cloud
3. Create a new app and link your GitHub repo.
4. Set `cna_score_dashboard.py` as the main file.
